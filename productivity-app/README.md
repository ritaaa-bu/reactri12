# 🚀 Продуктивность - Мобильное приложение для повышения личной эффективности

Приложение для управления повседневными активностями и повышения продуктивности.

## Функционал

### 1. 🎯 Трекер привычек
- Создание и отслеживание ежедневных привычек
- Подсчет серии дней (streak) для мотивации
- Отметка выполненных привычек

### 2. 📋 Менеджер задач
- Список задач на день
- Приоритеты задач (высокий, средний, низкий)
- Отметка выполненных задач

### 3. ⏱️ Pomodoro таймер
- Режим работы (25 минут)
- Короткий перерыв (5 минут)
- Длинный перерыв (15 минут)
- Старт/пауза/сброс таймера

### 4. 📊 Статистика
- Прогресс выполнения привычек и задач
- Общий стрик всех привычек
- Процент продуктивности за день
- Визуальный прогресс-бар

## Технологии

- **React Native** - кроссплатформенная разработка
- **Expo** - фреймворк для упрощения разработки
- **Hooks** - useState, useEffect для управления состоянием

## Запуск проекта

### Требования
- Node.js 16+ 
- npm или yarn
- Expo CLI (опционально)

### Установка зависимостей
```bash
cd productivity-app
npm install
```

### Запуск
```bash
# Запуск сервера разработки
npm start

# Запуск на Android
npm run android

# Запуск на iOS (требуется macOS)
npm run ios

# Запуск в веб-браузере
npm run web
```

### Сканирование QR-кода
После запуска `npm start` отобразится QR-код. Отсканируйте его:
- **iOS**: Камерой или приложением Expo Go
- **Android**: Приложением Expo Go

## Особенности UI/UX

- **Нижняя навигационная панель** - 4 вкладки для быстрого переключения
- **Floating Action Button (FAB)** - быстрое добавление привычек/задач
- **Модальные окна** - удобный ввод новых данных
- **Адаптивный дизайн** - работает на разных размерах экранов
- **Цветовая схема** - приятные цвета для длительной работы
