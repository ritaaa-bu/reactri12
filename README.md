# 📊 Продуктивность - Инструмент повышения личной эффективности

Веб-приложение для управления личной продуктивностью, разработанное для дипломного проекта.

## 🚀 Возможности

### 1. Трекер привычек
- Добавление и удаление привычек
- Подсчет серий (streak) - дней подряд выполнения
- Визуальное отслеживание прогресса
- Цветовая кодировка для каждой привычки

### 2. Менеджер задач
- Создание задач с приоритетами (высокий, средний, низкий)
- Отметка выполненных задач
- Цветовая индикация приоритетов
- Удаление задач

### 3. Таймер Pomodoro
- Режим работы (25 минут)
- Короткий перерыв (5 минут)
- Длинный перерыв (15 минут)
- Старт/пауза/сброс таймера

### 4. Статистика продуктивности
- Прогресс выполнения привычек за день
- Прогресс выполнения задач
- Визуальные индикаторы прогресса

## 💾 Хранение данных

Все данные сохраняются в localStorage браузера, что обеспечивает:
- Сохранение данных между сеансами
- Работу без серверной части
- Конфиденциальность (данные не покидают устройство)

## 🛠️ Технологии

- **React** - библиотека для создания пользовательских интерфейсов
- **Vite** - современный сборщик проектов
- **CSS3** - стилизация с градиентами и анимациями
- **localStorage API** - клиентское хранение данных

## 📦 Установка и запуск

### Требования
- Node.js (версия 16 или выше)
- npm

### Установка зависимостей
```bash
npm install
```

### Запуск в режиме разработки
```bash
npm run dev
```

### Сборка для продакшена
```bash
npm run build
```

### Предварительный просмотр сборки
```bash
npm run preview
```

## 🎨 Дизайн

Приложение использует современный дизайн с:
- Градиентным фоном
- Полупрозрачными элементами с эффектом blur

Приложение полностью адаптировано для:

- Планшетов

## 🔒 Конфиденциальность

- Все данные хранятся локально в браузере
- Нет серверной части и базы данных
- Данные не передаются третьим сторонам
