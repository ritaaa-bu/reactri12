# 🚀 Продуктивность - Веб-приложение для повышения личной эффективности

## Описание проекта

Веб-приложение "Продуктивность" разработано как инструмент для повышения личной продуктивности на основе повседневных активностей. Приложение реализовано в виде одностраничного сайта (Single Page Application) с использованием чистого HTML, CSS и JavaScript.

## Функциональные возможности

### 1. Трекер привычек 📊
- Добавление новых привычек
- Подсчет серий выполнения (streak counter)
- Отметка выполнения за текущий день
- Автоматический сброс ежедневного прогресса
- Визуальное отображение прогресса

### 2. Менеджер задач ✅
- Создание задач с указанием приоритета
- Три уровня приоритета: высокий, средний, низкий
- Цветовая кодировка приоритетов
- Отметка выполненных задач
- Удаление задач

### 3. Pomodoro Таймер ⏱️
- Режим работы (25 минут)
- Короткий перерыв (5 минут)
- Длинный перерыв (15 минут)
- Функции: старт, пауза, сброс
- Подсчет завершенных сессий за день

### 4. Статистика продуктивности 📈
- Прогресс выполнения привычек за сегодня
- Статистика выполненных задач
- Общая серия всех привычек
- Количество завершенных Pomodoro сессий

## Технические особенности

### Архитектура
- **Одностраничное приложение** - весь код в одном файле
- **Vanilla JavaScript** - без сторонних библиотек и фреймворков
- **LocalStorage** - сохранение данных в браузере пользователя
- **Адаптивный дизайн** - работает на всех устройствах

### Хранение данных
Все данные сохраняются в localStorage браузера:
- `habits` - массив привычек
- `tasks` - массив задач
- `pomodorosToday` - счетчик Pomodoro за день
- `lastResetDate` - дата последнего сброса

### Структура кода

#### HTML структура:
```html
<!DOCTYPE html>
<html lang="ru">
<head>
    <!-- Мета-теги и стили -->
</head>
<body>
    <div class="container">
        <!-- 4 карточки с функционалом -->
    </div>
    <script>
        // Логика приложения
    </script>
</body>
</html>
```

#### Основные функции JavaScript:
- `saveData()` - сохранение данных в localStorage
- `updateStats()` - обновление статистики
- `renderHabits()` - отрисовка списка привычек
- `renderTasks()` - отрисовка списка задач
- `addHabit()`, `toggleHabit()`, `deleteHabit()` - управление привычками
- `addTask()`, `toggleTask()`, `deleteTask()` - управление задачами
- `startTimer()`, `pauseTimer()`, `resetTimer()` - управление таймером

## Запуск приложения

### Способ 1: Открытие файла в браузере
1. Откройте файл `index.html` в любом современном браузере
2. Приложение готово к использованию

### Способ 2: Через VS Code Live Server
1. Установите расширение "Live Server" в VS Code
2. Откройте папку с проектом
3. Нажмите правой кнопкой на `index.html` → "Open with Live Server"

### Способ 3: Локальный сервер
```bash
# Python 3
python -m http.server 8000

# Node.js (если установлен http-server)
npx http-server

# Затем откройте http://localhost:8000
```

## Демонстрация для диплома

### Сценарий демонстрации:

1. **Трекер привычек:**
   - Добавить новую привычку (например, "Чтение")
   - Отметить выполнение
   - Показать увеличение серии

2. **Менеджер задач:**
   - Создать задачу с высоким приоритетом
   - Создать задачу с низким приоритетом
   - Отметить задачу как выполненную
   - Показать цветовую дифференциацию

3. **Pomodoro Таймер:**
   - Запустить таймер работы
   - Показать переключение режимов
   - Продемонстрировать паузу и сброс

4. **Статистика:**
   - Показать обновление статистики в реальном времени
   - Продемонстрировать сохранение данных
